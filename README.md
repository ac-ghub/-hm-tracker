# Half Marathon Tracker — Sub 2:00:00

22-week half marathon training plan tracker. Race day: 10 October 2025.

Built with Next.js 14 (App Router). Click any session to mark it complete — progress saves to localStorage.

## Deploy to Vercel

### Option 1 — Vercel CLI (fastest)
```bash
npm i -g vercel
vercel
```

### Option 2 — GitHub + Vercel dashboard
1. Push this folder to a GitHub repo
2. Go to vercel.com → New Project → Import the repo
3. Framework will be auto-detected as Next.js
4. Click Deploy — done

## Run locally
```bash
npm install
npm run dev
```
 
Open [http://localhost:3000](http://localhost:3000).
